## 依赖环境

1. PHP7.0 版本及以上（低版本和7.4版本未做兼容处理！）

## 安装

```text
composer require dtapps/laravel-library ^7.x-dev -vvv
```

## 更新

```text
composer update dtapps/laravel-library -vvv
```

## 删除

```text
composer remove dtapps/laravel-library -vvv
```
